# A wave simulator in Rust with Bevy

This is just a POC for further studies with visual waves

## How to run the program

### Prerequisites
As Bevy Engine is used, check their site to install what you need on your machine (https://bevy.org/)

### Commands
As usual with rust, I recommend `cargo run --release`.

## What you get
* You should see a window popping.
* Left click where you want to spawn the wave.
* Right click : press and hold the right button to draw obstacles.
